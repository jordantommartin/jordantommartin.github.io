Goal Oriented Action Planning Sim

Personal Programming Project / Unity

The personal programming project is an individual effort to explore a specific area of video game programming that I am interested in. I have chosen to focus on Goal Oriented Action Planning for artificial intelligence in games. Not only will I explore how to create a GOAP AI system, but I will apply this system to a theme park tycoon game where you can easily see the goals of each agent in the simulation.

Project Objectives

  • Create a GOAP AI system in Unreal
  • Integrate the GOAP AI system into an agent-based simulation
  • Show clear visualizations in real-time of the underlying GOAP AI system
  • Create a theme park tycoon game out of the GOAP AI system
